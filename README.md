# Weather App

Aplicación web moderna para la consulta de información meteorológica en tiempo real. Proporciona datos actuales y pronósticos del clima para cualquier ubicación a nivel mundial.

## Tecnologías

- **React** - Framework de interfaz de usuario
- **Bun** - Runtime y gestor de paquetes

## Características principales

- Consulta de condiciones meteorológicas actuales
- Pronósticos del tiempo por ubicación
- Interfaz de usuario intuitiva y minimalista
- Diseño responsive adaptado a dispositivos móviles y de escritorio

## Requisitos del sistema

- Bun 1.3 o superior

## Instalación

Instale las dependencias del proyecto:

```bash
bun install
```

Inicie el servidor de desarrollo:

```bash
bun run dev
```

La aplicación estará disponible en: `http://localhost:5173`

## Configuración

Este proyecto utiliza la API de [Open-Meteo](https://open-meteo.com), un servicio gratuito que no requiere autenticación ni clave de API. Para más información sobre los endpoints disponibles y parámetros de consulta, consulte la [documentación oficial](https://open-meteo.com/en/docs).
